# scottycore v0.1.7 — httpx runtime dep

Point release after consumer-app deploy testing.

## Bug fix

* `httpx` was listed under `[project.optional-dependencies.dev]` but
  `scottycore.services.backup.sinks.scottydev` imports it at top-level.
  Any consumer app that imported `scottycore.api.v1.router` (via the
  normal startup path) crashed with `ModuleNotFoundError: httpx` when
  httpx wasn't already installed as an app-level dependency.

  Fixed by promoting httpx to a runtime dependency in the base
  `[project.dependencies]` list.

## Upgrade

Pin bump only. No schema migration.
